English

Microsoft Visual Studio International Pack 1.0 Service Release 1 Readme File 

In this service release, the following two libraries contain fixes.  Other libraries are unchanged. 

Japanese Kana Conversion Library 
Fixed issues with Kana conversion where particular sequences did not get converted properly.

Japanese Yomi Auto-Completion Library 
Fixed an issue where the candidate window was truncated during Auto-Completion.

Microsoft Visual Studio International Pack 1.0 Service Release 1 讀我檔案

在這一個 service release 中, 我們只修正了下列兩個程式庫.  其他的程式庫則維持原樣. 

日文 Kana 轉換類別庫 (Japanese Kana Conversion Library)
修正了在特定順序下的 Kana 字串無法正確轉換的問題.

日文 Yomi 自動完成類別庫 (Japanese Yomi Auto-Completion Library )
